## Getting Started

### Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/fedemarolla/ChatProject.git
   cd TP-FINAL-UTN
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm start
   ```

4. Open your browser at:

   ```bash
   http://localhost:5173
   ```
